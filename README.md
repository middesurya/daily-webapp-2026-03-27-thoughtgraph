# ThoughtGraph — Personal Knowledge Graph

An interactive personal knowledge management tool that lets you capture ideas as nodes and automatically discovers connections between them using text similarity. Beautiful force-directed graph visualization powered by D3.js.

## Features

- **Add Thoughts** — Quick capture of ideas, notes, snippets with optional tags
- **Auto-Linking** — Automatic discovery of connections using TF-IDF text similarity
- **Interactive Graph** — D3.js force-directed graph with zoom, pan, drag
- **Tag System** — Color-coded categories for organization
- **Search & Filter** — Full-text search and tag filtering
- **Import/Export** — JSON export/import for data portability
- **Dark/Light Mode** — Toggle between themes
- **Keyboard Shortcuts** — `N` to add, `/` to search, `Esc` to close

## Tech Stack

- Single HTML file — no build tools, no framework
- D3.js v7 for force-directed graph visualization
- Vanilla JS TF-IDF implementation for text similarity
- localStorage for persistence
